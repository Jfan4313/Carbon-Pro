## 版本说明

### 语义化版本号 (Semantic Versioning)

本项目采用 `MAJOR.MINOR.PATCH` 格式：

- **MAJOR** (主版本号): 不兼容的 API 修改
- **MINOR** (次版本号): 向下兼容的新功能
- **PATCH** (修订号): 向下兼容的问题修复

### 变更类型标识

- `新增` (Added): 新功能
- `修复` (Fixed): Bug 修复
- `优化` (Changed): 功能改进
- `废弃` (Deprecated): 即将移除的功能
- `移除` (Removed): 已移除的功能
- `安全` (Security): 安全相关修复
